
# Admission Analyzer (кейс «Анализ поступления»)

Мини-проект на Python: генерация конкурсных списков, загрузка/обновление в БД, визуализация (веб-интерфейс), расчет проходных баллов с учетом согласий и приоритетов, формирование PDF-отчетов.

## Стек
- Python 3.10+
- Flask (веб-интерфейс)
- SQLite + SQLAlchemy (БД)
- pandas (загрузка CSV)
- reportlab + matplotlib (PDF + графики)

## Быстрый старт
```bash
pip install -r requirements.txt
python generate_lists.py      # сгенерирует 16 CSV в папку data/
python app.py                 # запустит веб-приложение
```

Открыть: http://127.0.0.1:5000

## Структура CSV
Колонки строго по ТЗ:
`id, consent, priority, phys, rus, math, indiv, total`

## Как пройти испытания
### Испытание №1
1) Убедитесь, что есть 16 CSV (4 дня × 4 ОП) в `data/`.
2) В веб-приложении нажмите «Очистить БД», затем загрузите файлы 01.08 по всем ОП.
3) Загрузите 02.08 — продемонстрируйте, что часть записей удалена/обновлена, появились новые.

### Испытание №2
- После загрузки 01.08 нажмите «Рассчитать» — получите «НЕДОБОР» (согласий меньше мест).
- После загрузки 02.08 / 03.08 / 04.08 — покажите изменение проходных баллов и перемещение абитуриентов из-за приоритетов.

### Испытание №3
- Нажмите «PDF-отчет» для каждого дня.
